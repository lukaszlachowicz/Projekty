Exploratory Analysis of 2019 Sales Data

Purpose of the project
The main objective of this project was to conduct an exploratory analysis of sales data from 2019 in order to identify key trends in customer behavior, 
product popularity, revenue generation, and seasonality. By analyzing purchase patterns, regional sales distributions, and product combinations, the project 
aimed to extract actionable insights that could help businesses improve their sales strategies, optimize product offerings, and better understand consumer preferences.

Data
Source: https://github.com/KeithGalli/Pandas-Data-Science-Tasks/tree/master/SalesAnalysis/Sales_Data
Scope: January 2019 – December 2019
Format: CSV (12 files merged into a single dataset)

Technologies  
- Python – main programming language used for data processing and analysis  
- Pandas – data manipulation and cleaning  
- Matplotlib & Seaborn – data visualization  
- Jupyter Notebook – interactive environment for analysis and presentation

Results  
Most frequently ordered products: Batteries, chargers, and headphones  
Highest revenue: MacBook Pro and other laptops, as well as smartphones  
Seasonality: Most orders placed in December, October, and November  
Order timing: Peaks around 12:00 PM and 7:00 PM  
Regional trends: California had the highest number of orders and revenue; Maine the lowest  
Average shopping cart: 1.17 products, average total price $193.30  
Buying patterns: Frequent bundles of phones with accessories (e.g., iPhone + charging cable)
